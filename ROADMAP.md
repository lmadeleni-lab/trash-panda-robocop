# Roadmap

## Near Term

- add image ingest and snapshot review endpoints
- plug in a real wildlife detector backend behind the bounded perception interface
- improve nightly summaries with richer return and nuisance analysis
- add hardware-in-the-loop smoke tests for Raspberry Pi deployments

## Mid Term

- add privacy-aware image retention and redaction policies
- support deployment-specific calibration workflows for gates and protected zones
- add actuator health checks and watchdog behavior for outdoor nodes
- add optional operator dashboards for local review

## Long Term

- stronger adaptation logic based on bounded strategy outcomes
- standardized hardware reference builds for backyard gate and pool-boundary installs
- field-tested deployment recipes for Pi-based weatherproof nodes

## Explicit Non-Goals

- harming animals
- trapping or cornering wildlife
- freeform external-agent hardware control
- pursuit or mobile chase platforms
