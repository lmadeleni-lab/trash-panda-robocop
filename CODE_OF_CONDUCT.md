# Code of Conduct

## Our Standard

We want this project to be welcoming, practical, and respectful. Contributors should help make the repo a place where people can ask basic questions, share ideas, and improve the work without being dismissed or embarrassed.

Examples of positive behavior:

- being respectful and constructive
- giving actionable technical feedback
- assuming good intent while still being clear about risks
- helping new contributors understand context

Examples of unacceptable behavior:

- harassment or intimidation
- personal attacks
- exclusionary or demeaning language
- deliberately unsafe or harmful suggestions presented as acceptable project direction

## Enforcement

Project maintainers may remove, edit, or reject contributions or comments that violate this code of conduct or the project’s humane-safety scope.

If you experience or witness unacceptable behavior, please contact the maintainers privately through the security or maintainer contact path listed in [SECURITY.md](/Users/laurent/Development/trash-panda-robocop/SECURITY.md).
